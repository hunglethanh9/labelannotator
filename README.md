# radiologyannotation
Radiology Annotation Tool

## Requirements

 - [Node v7.6+](https://nodejs.org/en/download/current/) or [Docker](https://www.docker.com/)
 - [Npm](https://www.npmjs.com/)

## Getting Started

Install dependencies:
```bash
npm install
```

Set environment variables:

```bash
cp .env.example .env
```

## Running Locally

```bash
gulp server
```
